# Car Listing Admin Dashboard

##Features

- Clerk Authentication
- Listing Actions (edit,Approve,reject)
- Audit log of Actions
- SSR with 'getServerSideProps'
- ShadCN UI & TailwindCSS for frontend styles and components
  -Pagination for listings

## Tech Stack

- Next.js (Page Router)
- Prisma ORM
- PostgresSQL (Neon Postgres Database)
- Clerk (Authentication and user management)
- ShadCN UI (UI Component)
- Axios
- TailwindCSS
